Color Principal

#1B1B1B

El color principal elegido es este tipo de negro, llamado Nero, variante ligeramente más clara que el negro convencional. El motivo de esta elección se basa en simplemente la comodidad que proporciona a la hora de visitar la pagina, ya que este color no dispone de tonos brillantes y pesados a la vista.

Colores Secundarios

#0CF

#EFC621

El color azulado llamado Vivid Sky Blue, es usado en mi web para los elementos los cuales necesito que sean destacados. Por ejemplo, podemos encontrar dicho color en seleccionados seleccionados como botones o listas, o fechas.

El color amarillo llamado Kode Warna Hex, es usado en mi web para aquellos elementos que se refieren a categorias de contenido o para indicar que un elemento es seleccionable, normalmente aparece una etiqueta debajo de cada imagen con esa caracteristica.

Color Textos

#FFFFFF

Este color es el blanco común, utilizado para titulos y textos. La elección de este color se basa simplemente en que el blanco es muy detacable encima de colores oscuros como nuestro color principal.
Cabe destacar que para que la web no se vea muy sobrecargada de texto, hemos incluido una opacidad al color blanco, acabando en ser un color mas grisáceo.
