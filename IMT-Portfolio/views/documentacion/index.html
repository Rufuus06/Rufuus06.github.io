Tipografía

Viernes 14 de Mayo

Abel es una interpretación moderna del Sans Serif condensado de lados planos. Originalmente utilizado para titulares de periódicos y carteles, este estilo también se puede utilizar para texto en la web.
Sus terminales en ángulo y vástagos puntiagudos le dan suficiente estilo para ser único en tamaños de pantalla, mientras que su peso mono todavía funciona bien en tamaños de texto más pequeños.

Color Principal

#1B1B1B

El color principal elegido es este tipo de negro, llamado Nero, variante ligeramente más clara que el negro convencional. El motivo de esta elección se basa en simplemente la comodidad que proporciona a la hora de visitar la pagina, ya que este color no dispone de tonos brillantes y pesados a la vista.

Botones

Viernes 14 de Mayo
Boton

Este modelo de botón es el que se utiliza en esta web, presenta un fondo transparente y bordes grises.
A la hora de situarse encima del elemento, éste cambia su fondo a negro y cambia la letra de su interior al color azul mencionado en Colores Secundarios.

Colores Secundarios

#0CF

#EFC621

El color azulado llamado Vivid Sky Blue, es usado en mi web para los elementos los cuales necesito que sean destacados. Por ejemplo, podemos encontrar dicho color en seleccionados seleccionados como botones o listas, o fechas.

El color amarillo llamado Kode Warna Hex, es usado en mi web para aquellos elementos que se refieren a categorias de contenido o para indicar que un elemento es seleccionable, normalmente aparece una etiqueta debajo de cada imagen con esa caracteristica.

Logo

Viernes 14 de Mayo

El logo de la web esta constituido con las letras de mi nickname Rufuus.
Se trata de una superposición de todas las letras de la palabra hasta formar la imagen que veis arriba.
El objetivo del diseño es que resulte sencillo y limpio, por ese motivo no dispone de colores, simplemente lineas negras.

Unity

Viernes 14 de Mayo

Para desarrollar mi juego, he utilizado el programa Unity, el cual ofrece un sistema adaptado para la creación de videojuegos.
Además, para el diseño de los Sprites, he utilizado Photosohop, herramienta muy útil para crear o editar cualquier imagen o documento.

Color Textos

#FFFFFF

Este color es el blanco común, utilizado para titulos y textos. La elección de este color se basa simplemente en que el blanco es muy detacable encima de colores oscuros como nuestro color principal.
Cabe destacar que para que la web no se vea muy sobrecargada de texto, hemos incluido una opacidad al color blanco, acabando en ser un color mas grisáceo.
